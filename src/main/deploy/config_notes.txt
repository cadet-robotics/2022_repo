Controls:
    driver joystick
        ID of the controller that the Driver Joystick is attached to
    dj slider
        ID of the slider axis on the controller
    dj intake manual
        ID of the trigger, activating the front ball intake

        General override key, going to be used to override automatic functions and commands, and to
        override checks and sensor data.
